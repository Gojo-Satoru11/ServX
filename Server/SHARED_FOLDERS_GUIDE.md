## ğŸ› Troubleshooting

### "No other users registered"
- Need at least 2 users total to share
- Create another account or invite someone
- Can't share with yourself

### Can't see shared folder
- Make sure you're logged in
- Check if you're in the share list
- Ask owner to verify sharing

### Can't delete folder
- Only owner can delete folders
- You can delete individual files
- Contact folder owner

### Upload fails
- Check file is under 2GB
- Verify file type is allowed
- Ensure stable connection
